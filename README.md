# Raparigage Command Line Sound

> Alô inspetô!

## Compatibility

- Linux
- Windows
- Mac

## Installing
Install globally:

    npm install -g raparigage

## Running
Run command:

    raparigage

Or import in NodeJS script:

    import raparigage from 'raparigage';

    raparigage();
    
## Running with [npx](https://www.npmjs.com/package/npx)
Run command:

    npx raparigage
